计算总览
1）权重层级
一级权重（W1）：如“数字化基础50%”→ 0.5
二级权重（W2）：如“设备系统40%”“规划管理50%”→ 0.4、0.5
采集项权重（W3）：即该二级下的分项占比，如“网络建设40%”“设备数字化30%”“设备联网率30%”→ 0.4/0.3/0.3；或“网络安全50%/数据安全50%”→ 0.5/0.5；有的为 100% → 1.0
这三层权重直接来自表头文本（已带百分号）。
2）题型与得分
单选：按“得分说明”的顺序映射分值，例如“从上至下：0,1.5,3,4.5,6”。
多选：每选项固定加分（如“每项 2 分，最高 8 分”），或“每项 1.25 分，最高 5 分”等。
趋势题（如“降低/持平/增加”）：按“降低不得分/持平 X 分/增加 Y 分”映射。
每道题有明确 max_score（上限），用于规范化。
3）题目贡献值（规范化）
令某题原始得分为 s_i，该题上限为 max_i，则：
norm_i = s_i / max_i            # 0~1
contrib_i = norm_i * W1 * W2 * W3  # 题目对“总分(满分=1.0)”的贡献
如果你要把总分显示为 100 分制，则最终总分 Score100 = sum(contrib_i) * 100。

编码计算（伪代码）
def compute_total_score(responses, config):
    # responses: {item_id: response_obj}
    # response_obj for "single":
    #   - by index: {"type": "single", "index": 0-based}
    #   - by label: {"type": "single", "label": "增加/持平/降低"}
    # response_obj for "multi":
    #   - {"type": "multi", "selected_count": int}

    total = 0.0
    for dim in config["dimensions"]:
        W1 = dim["weight"]
        for sub in dim["subdimensions"]:
            W2 = sub["weight"]
            for item in sub["items"]:
                item_id = item["id"]
                W3 = item["weight"]
                sc = item["scoring"]
                raw = 0.0
                max_i = sc["max_score"]

                r = responses.get(item_id)
                if r is None:
                    raw = 0.0
                elif item["type"] == "single":
                    if "map" in sc:
                        # index mapping
                        idx = r.get("index", 0)
                        raw = sc["map"][idx]
                    elif "map_label_to_score" in sc:
                        label = r.get("label")
                        raw = sc["map_label_to_score"].get(label, 0.0)
                elif item["type"] == "multi":
                    cnt = r.get("selected_count", 0)
                    raw = min(cnt * sc["per_option"], max_i)

                norm = 0.0 if max_i == 0 else (raw / max_i)
                contrib = norm * W1 * W2 * W3
                total += contrib

    # 转为 100 分制
    return round(total * config["metadata"]["score_scale"], 2)
问答数据到分值的映射要点
单选（按顺序）：保存“所选选项的下标 index”，从 0 开始；代码按 map[index] 取分。
例：设备数字化率选第 4 档（“40–60%”）→ index=3 → 得分 4.5。
单选（按标签）：保存 label 为“增加/持平/降低”。
多选：只需要保存 selected_count（勾选数量），程序用 per_option × count 再截断到 max_score。
升级判定（2→3）
def check_upgrade_2_to_3(scene_count, constrained_scene_count, config):
    rule = config["upgrade_rules"]
    return (scene_count >= rule["min_scenes_total"] and
            constrained_scene_count >= rule["min_constrained_scenes"])
