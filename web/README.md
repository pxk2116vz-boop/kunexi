# 数字化问卷系统

基于 React + TypeScript + Vite 构建的企业信息化调研问卷应用。系统内置题库、权重模型与实时评分逻辑，可支持选项备注、场景升级判定、数据导入导出等功能，适用于科技感调研表单的快速部署。

## 功能亮点

- 📊 **实时算分引擎**：按照《计算逻辑.md》配置的三层权重与分值映射自动计算总分与维度贡献，支持 100 分制展示与雷达图分析。
- 📝 **选项备注体系**：每个选项均可展开备注文本框，记录补充信息并持久化到本地存储。
- 🧭 **结构化题库渲染**：维度/子维度/题目配置统一管理，自动渲染单选、多选、跟进输入项等控件。
- 🚀 **升级判定提醒**：结合上线场景与约束性场景数量，实时提示是否满足等级升级条件及缺口。
- 💾 **数据管理**：支持一键导出/导入 JSON 快照与清空答卷，方便跨设备迁移与复用。
- 🎨 **科技感界面**：渐变发光背景、玻璃拟态卡片与动态进度条营造数字化视觉体验。

## 本地运行

```bash
npm install
npm run dev
```

默认端口为 `4173`，可通过 `-- --host 0.0.0.0 --port <port>` 指定地址与端口。生产构建使用：

```bash
npm run build
npm run preview
```

## 目录结构

```
web/
├── src/
│   ├── components/          # 通用界面组件
│   ├── config/              # 问卷及基础字段配置
│   ├── store/               # Zustand 状态与持久化
│   ├── utils/               # 算分与升级判定逻辑
│   └── types/               # 类型定义
└── public/                  # 静态资源
```

## 数据导入/导出说明

- **导出**：点击“导出 JSON”按钮，系统会下载当前答卷快照（含基本信息、作答、备注、场景数据）。
- **导入**：点击“导入 JSON”选择历史快照文件，可恢复之前的答题进度。
- **清空**：选择“清空答卷”将删除本地存储的全部问卷数据。

## 自定义扩展

- 在 `src/config/questionnaire.ts` 中新增或调整题目、权重与选项即可扩展问卷结构。
- `src/utils/scoring.ts` 提供算分与维度拆解函数，可用于后端接口或报表导出。
- 通过 `SceneSection` 中的配置，可拓展更多成效指标、支持诉求等额外字段。
